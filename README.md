# PYTHON-STAR-PATTERN-CODE
STAR PATTERN CODE REPOSITORY FOR PYTHON
# 1. Create Rectangle pattern:<br>

```python
for i in range(n):
  for j in range(i+1):
    print("*",end=" ")
  for j in range(i,n):
    print("*",end=" ")
  print(" ")
```
## OUTPUT:

```
* * * * * *  
* * * * * *  
* * * * * *  
* * * * * *  
* * * * * *
```

# 2. Create increasing triangle  pattern:<br>

```python
for i in range(n):
    for j in range(i+1): # loop for increasing  triangle
        print("*",end=" ") 
    print(" ")

```
## OUTPUT:

```
*
* *
* * *
* * * *
* * * * *
```
